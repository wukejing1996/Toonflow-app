## 二、全局色彩盘（风格基线，不是硬锁）

> 目标：统一审美而非限制创作。除「硬约束色」外，其余颜色默认优先使用，可在合理范围内偏移。

### 色彩使用层级

| 层级 | 约束强度 | 说明 |
|---|---|---|
| L1 硬约束 | 高 | 统一整体“明亮柔和”的色彩逻辑：高明度、低到中饱和 |
| L2 软约束 | 中 | 服饰/配件/场景优先从推荐色盘选取 |
| L3 例外机制 | 低 | 情绪高潮/节日镜头允许局部更鲜艳点缀，但不可荧光霓虹 |

| 序号 | 色名 | 色值 | 用途 |
|---|---|---|---|
| C1 | 奶油白 | #FFF6E8 | 背景基底、纸张感 |
| C2 | 蜜桃粉 | #F7C6C7 | 脸颊红晕、温暖点缀 |
| C3 | 柠檬黄 | #FFE08A | 阳光、装饰、温馨氛围 |
| C4 | 薄荷绿 | #BFE3C0 | 草地、植物、清新环境 |
| C5 | 天空蓝 | #A9D4F5 | 天空、水面、清爽背景 |
| C6 | 薰衣草紫 | #D7C7F2 | 梦幻、夜晚轻氛围 |
| C7 | 焦糖棕 | #B57A5A | 木质、地面、暖中性色 |
| C8 | 珊瑚橙 | #FF9A7A | 小面积强调、配件点缀 |
| C9 | 浅灰 | #D9D9DF | 阴影、边界过渡、中性平衡 |
| C10 | 草绿色 | #7CCB7A | 重点植被、点缀绿 |

### 色温/饱和/对比建议

| 参数 | 值 | 说明 |
|---|---|---|
| 整体色温 | 偏暖（推荐） | 童书绘本的温暖感 |
| 对比度 | 中低 | 保持柔和，不要硬对比 |
| 饱和度 | 中低到中 45–70% | 避免荧光与脏灰 |

## 三、全局约束规则

### 必守规则（所有技能继承）

| 编号 | 规则 |
|---|---|
| R1 | 必须包含「2D cartoon illustration / cute animal / clean line art」锚定词 |
| R2 | 必须包含「soft pastel colors」或同义描述 |
| R3 | 必须声明「simple shading / soft shadow」轻阴影（不可重阴影） |
| R4 | 角色造型必须圆润可爱、表情清晰，避免尖锐恐怖造型 |

### 严禁项（所有技能继承）

| 编号 | 严禁内容 |
|---|---|
| X1 | 严禁写实/照片级真实感/真实摄影术语（photorealistic/RAW photo 等） |
| X2 | 严禁暗黑恐怖、血腥暴力、猎奇拟真细节 |
| X3 | 严禁高饱和荧光霓虹/赛博朋克霓虹灯海 |
| X4 | 严禁重阴影、硬光强对比导致“脏黑” |

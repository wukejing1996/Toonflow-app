# 可爱动物角色设定图生成 · 约束手册

---

## 一、角色设计原则

1. **圆润可爱** — 以圆形/椭圆为主的结构语言，减少尖锐折线。
2. **表情清晰** — 眼睛与嘴型是情绪锚点，需清晰、友好、可读。
3. **比例统一** — 默认“大头小身”，四视图体型与面部特征一致。
4. **细节克制** — 花纹/配件点到为止，避免信息噪声。

---

## 二、动物通用面部约束

| 部位 | 约束 | 提示词 |
|---|---|---|
| 头部 | 大头、圆润轮廓 | big round head, rounded silhouette |
| 眼睛 | 大而明亮，黑色或深棕瞳 | big bright eyes, dark pupils |
| 鼻口 | 小鼻子，小嘴，微笑友好 | small nose, small mouth, gentle smile |
| 眉眼 | 眉形简化，用眼型表达情绪 | simplified eyebrows, expressive eyes |
| 线稿 | 线条干净均匀 | clean line art, smooth outline |

---

## 三、四视图设定图规范

### 视图定义

| 位置 | 视图 | 角度 | 要求 | 提示词 |
|---|---|---|---|---|
| 左一 | 表情特写 | 正面 | 眼睛/嘴型/腮红清晰 | face closeup |
| 左二 | 正视图 | 0° | 全身完整展示 | front view, full body |
| 右二 | 侧视图 | 90° | 轮廓可读，尾巴/耳朵结构清晰 | side view |
| 右一 | 后视图 | 180° | 背部花纹/尾巴清晰 | back view |

### 画面规范

| 项目 | 约束 |
|---|---|
| 布局 | 同一画面从左至右并排四视图 |
| 背景 | 纯净浅色背景（推荐 C1 奶油白） |
| 姿态 | 中立站姿，避免复杂动作 |
| 阴影 | 仅柔和轻阴影（脚下浅投影） |
| 一致性 | 四视图造型一致，花纹位置一致 |

---

## 四、提示词模板

```
2D cartoon cute animal character design sheet, character turnaround,
clean line art, smooth outline, soft pastel colors,
cute proportions (big head, small body), friendly expression,
simple shading, soft shadow, high key lighting,
same image left-to-right: face closeup + front view full body + side view + back view,
plain light background, minimal details,
high resolution, crisp lines,
no text, no watermark
```

---

## 五、约束规则

### 必守

| 编号 | 规则 |
|---|---|
| R1 | 必须是可爱动物四视图（特写+正视+侧视+后视） |
| R2 | 必须包含干净线稿与柔和粉彩色块 |
| R3 | 必须只有轻阴影（脚下浅投影） |
| R4 | 背景必须纯净浅色 |

### 严禁

| 编号 | 严禁 |
|---|---|
| X1 | 写实/照片级真实感 |
| X2 | 重阴影、黑暗恐怖氛围 |
| X3 | 荧光霓虹与赛博朋克调色 |
| X4 | 复杂背景与大量道具干扰 |

# 可爱道具设定图生成 · 约束手册

---

## 一、道具设计原则

1. **用途清晰** — 一眼能看出是食物/玩具/日用品/小工具。
2. **形状圆润** — 道具轮廓尽量圆角、软萌，避免尖锐硬边。
3. **色彩柔和** — 以粉彩色块为主，小面积强调色即可。
4. **纯道具展示** — 画面中只能出现道具本身，严禁手部/人物。

---

## 二、多角度设定图规范（2×2）

| 位置 | 视图 | 要求 | 提示词 |
|---|---|---|---|
| 左上 | 正面图 | 造型可读 | front view |
| 右上 | 侧面图 | 厚度轮廓清晰 | side view |
| 左下 | 背面图 | 背面结构与花纹 | back view |
| 右下 | 细节特写 | 质感/图案/开合处 | detail closeup |

---

## 三、提示词模板

```
2D cartoon cute prop design sheet,
clean line art, soft pastel colors, simple shading, soft shadow,
2x2 grid: front view + side view + back view + detail closeup,
isolated prop only, plain light background,
high resolution, crisp lines,
no text, no watermark
```

---

## 四、约束规则

### 必守

| 编号 | 规则 |
|---|---|
| R1 | 必须为 2×2 四宫格多视图 |
| R2 | 必须为纯道具展示（无手无人物） |
| R3 | 干净线稿 + 粉彩色块 + 轻阴影 |

### 严禁

| 编号 | 严禁 |
|---|---|
| X1 | 写实/照片级真实感 |
| X2 | 重阴影与暗黑恐怖氛围 |
| X3 | 霓虹赛博朋克调色 |

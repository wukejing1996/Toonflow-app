# 2D 卡通可爱动物风格说明

本风格专为「2D 卡通插画 · 可爱动物」题材打造：以**干净线稿 + 柔和色块/轻阴影**为基础，以**童书绘本气质**为审美方向，强调角色的**亲和力、可读性与一致性**。

## 适用范围

- 可爱动物角色设定（单只/多只动物，四视图/转身图）
- 动物服饰与小配件衍生（围巾、背包、帽子等）
- 可爱道具设定（食物、玩具、日用品、小工具）
- 童书/动画向的自然场景（森林、草地、树屋、湖边、温馨小屋）
- 分镜与视频提示词（轻松治愈、温暖明亮）

## 风格关键词

- **卡通可爱比例**：大头小身、圆润轮廓、表情清晰
- **线条干净**：线稿柔和均匀，不用硬锐工业线
- **柔和配色**：低到中饱和的马卡龙/粉彩色系，整体高明度
- **轻质感**：少量柔和阴影与体积感，避免厚重写实光影

## 严禁内容

- 写实/照片级真实感、真实摄影风（real photo/RAW 等）
- 暗黑恐怖、血腥暴力、猎奇拟真器官细节
- 过度霓虹赛博朋克调色（高饱和荧光霓虹、强发光灯海）
- 过度复杂材质与重金属工业机械风（与“可爱动物”主题冲突）

---

本风格仅适用于「2D 卡通可爱动物」相关项目。如需写实动物纪录片风、暗黑怪诞风、重工业机甲风等，请勿使用本套提示词。

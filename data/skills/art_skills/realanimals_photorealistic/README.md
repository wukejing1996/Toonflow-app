# 写实动物摄影风格说明

本风格专为「写实动物 · 自然摄影/纪录片质感」题材打造：强调**真实动物解剖结构**、**毛发/羽毛微细节**、**自然光与真实景深**、以及**真实栖息地环境**。画面目标是“像自然摄影作品”，而不是插画或 3D 渲染。

## 适用范围

- 写实动物角色图（单只动物/多只动物）
- 动物特写（头部/眼神/毛发细节）
- 野生动物栖息地场景（森林、草原、雪原、海岸、湿地等）
- 纪录片分镜/镜头提示词（强调镜头语言与环境氛围）

## 风格关键词

- **照片级写实**：真实摄影观感、真实光学与景深
- **解剖准确**：骨骼比例、肌肉体块、关节结构符合真实动物
- **毛发细节**：毛流方向、层次、绒毛与硬毛区分清晰
- **自然环境**：真实地貌与植被，符合地理与季节

## 严禁内容

- 卡通/Q版/拟人化（穿衣、站立像人、夸张大眼等）
- 二次元/赛璐璐/扁平插画/矢量风
- 明显 3D 渲染/CG/游戏引擎风（Unreal/Unity/3D render 等）
- 霓虹赛博朋克调色、荧光高饱和灯海
- 血腥断肢、虐待动物等成人向暴力细节

---

本风格仅适用于「写实动物摄影」相关项目。如需卡通可爱动物、工业机甲、国风二次元等，请勿使用本套提示词。
